
# Vue Task App

  

Projeto de aplicativo de tarefas utilizando Vue.js.

  

## Versão Utilizada

  

Vue.js 3

  

## Instruções

  

Clone este repositório
```sh

git clone https://github.com/henriquetbp/vue-task-app.git

```
Instale as dependências
```sh

npm  install

```
Execute o projeto
```sh

npm  run dev

```
